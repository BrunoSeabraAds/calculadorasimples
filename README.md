# Calculadora em HTML

Projeto simples de uma calculadora desenvolvida com HTML, CSS e JavaScript.

## Sobre o projeto

Esta calculadora permite realizar operações matemáticas básicas de forma simples e intuitiva, com uma interface visual personalizada.

## Funcionalidades

- Soma
- Subtração
- Multiplicação
- Divisão
- Limpar resultado
- Apagar último número
- Interface responsiva e estilizada

## Tecnologias utilizadas

- HTML
